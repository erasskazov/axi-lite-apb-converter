APB4 - простой memory-mapped интерфейс из семейства AMBA.
AXI4-Lite - упрощенный AXI4 memory-mapped интерфейс, без поддержки  burst access и  transaction ID.
    - Требуется сделать параметризуемый преобразователь интерфейсов AXI4-Lite в APB и наоборот на основе axilite_to_apb.sv . 
    - Для подтверждения работоспособности необходимо написать простой тестбенч подтверждающий правильность работы интерфейса. 
    - Собрать проект в Vivado в качестве slave-а  можно использовать любой библиотечный IP c поддержкой используемого интерфейса.
    - Для уменьшения обьема передаваемых файлов  проект должен собираться TCL скриптом. 
    Дополнительно но не обязательно 
    - Использовать для описания SV interface для описания
